# SquidInk_1.0.1

A Mod for Factorio

## The Plan

### Items

1. Squid Plates - Metal plates Iron + Steel + Squid Ink

2. Squid Furnace - a better electric furnace

3. Uranium Processor - Creates Uranium 235 from 238 and Squid Ink

4. Squid Squeezer - Collects Squid Ink from fish
